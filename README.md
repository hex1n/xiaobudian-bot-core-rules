# 小不点 (Little Bit) 核心规约库

本仓库是“小不点”(Little Bit) 的核心资产库，存储了定义其行为准则、协作模式及运行逻辑的关键文档。

## 核心内容

本仓库主要包含以下核心维度：

- **性格 (SOUL)**：定义小不点的基本人格、价值观、沟通风格及核心使命。
- **团队规约 (TEAM)**：明确小不点在团队协作中的定位、权限边界及交互协议。
- **记忆 (MEMORY)**：存储长期记忆、历史决策及持续演化的知识图谱。
